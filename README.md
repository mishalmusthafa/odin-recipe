# odin-recipe

## This is my First odin project
This project is a recipe project which can showcase my Basic skills in my HTML 

In the project i have learned and created the basic elements like H1 ul ol img and a tags. I learned about the best practices about git commit, the 7 rules for the commit (Subject line and body). 

This project is recipe website. I have created a home page and 3 recipe pages. The 3 recipe pages are same template with different recipes. For the recipes I the format was an 
1. Heading with h1 tag for the Dish name.
2. Image of the dish with appropriate size in px
3. Heading with h2 tag for the recipes. list with <ul> for the recipes
4. Heading with h2 tag for the direction. List with <ol> for the direction
5. Committed the project with the best practice for the commit message and pushed to github
6. Published on github.
